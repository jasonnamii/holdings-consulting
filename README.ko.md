# holdings-consulting

> 🇺🇸 [English README](./README.md)

**홀딩스(지주회사) 범용 컨설팅 엔진. 신규설립·기존전환·운영최적화·승계설계·해외구조 5대 시나리오 대응.**

## 목표
지주회사 구조와 경영을 위한 종합 컨설팅 프레임을 제공합니다. 신규 지주회사 설립, 기존 사업 전환, 운영 최적화, 승계 계획, 해외 구조 설계 등 시나리오별 전략과 실행 프로토콜을 제공합니다.

## 주요 기능
- **5대 시나리오 프레임**: 신규설립, 기존전환, 운영최적화, 승계설계, 해외구조
- **구조화된 컨설팅 프로토콜**: 각 시나리오별 진단→전략설계→실행 로드맵
- **재무·법률 통합**: 절세 구조, 규제 준수, 자산 재구성
- **이해관계자 관리**: 이사회 거버넌스, 자회사 조율, M&A 통합
- **커스터마이징 플레이북**: 특정 그룹 구조와 목표에 맞게 프레임 적용
