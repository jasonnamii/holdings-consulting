# holdings-consulting

> 🇰🇷 [한국어 README](./README.ko.md)

**Holding company (conglomerate) universal consulting engine. Handles 5 major scenarios: new establishment, existing conversion, operational optimization, succession planning, and overseas structures.**

## Goal
This skill provides comprehensive consulting frameworks for holding company structures and management. Whether you're establishing a new holding company, converting an existing business, optimizing operations, planning succession, or designing overseas structures—this engine delivers scenario-specific strategies and implementation protocols.

## Key Features
- **5-Scenario Framework**: New establishment, existing conversion, operational optimization, succession planning, overseas structures
- **Structured Consulting Protocols**: Diagnosis → strategy design → implementation roadmap for each scenario
- **Financial & Legal Integration**: Tax efficiency, regulatory compliance, asset structuring
- **Stakeholder Management**: Board governance, subsidiary coordination, M&A integration
- **Customizable Playbooks**: Adapt frameworks to your specific conglomerate structure and goals
