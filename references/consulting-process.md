# 축7. 컨설팅 프로세스

스킬의 3단계 흐름(가이드 → 핑퐁 → 리포트)을 실행하는 방법론.

---

## 5단계 컨설팅 프로세스

```
①현황진단 → ②구조설계 → ③세무·법무검토 → ④실행계획 → ⑤실행·모니터링
```

### ①현황진단

핑퐁 Phase 1~3에서 수집. 상세 질문 목록 → `pingpong-questions.md` 참조.

**핑퐁 외 보조 정보 출처 (클라이언트에게 안내용):**
- 기업집단포털(공정거래위) — 기업집단 현황
- DART(전자공시) — 재무·지분·공시
- HOMETAX(국세청) — 세무 신고 현황

**수집 완료 판정:** 10대 판단분기점 해당 항목 전부 답변 확보 → 품질게이트(SKILL.md §5) 통과 시 ②로 진행.

### ②구조설계

**선택지 매트릭스:**

| 축 | 대안 A | 대안 B | 대안 C |
|----|--------|--------|--------|
| 전환 방식 | 인적분할 | 물적분할 | 현물출자+주식교환 |
| 구조 깊이 | 단층(지주+자회사) | 2층(중간지주) | 3층(다단계) |
| 금융 분리 | 금융·비금융 분리 | 통합 | 별도 금융지주 |
| 브랜드 | Branded House | House of Brands | Hybrid |

각 조합의 장단점 + 세무 영향 + 규제 적합성을 비교 테이블로 제시.

### ③세무·법무검토

| 검토 항목 | 시뮬레이션 내용 |
|----------|---------------|
| 배당세 | 지분율별 익금불산입 시나리오 (50%/30%/20%/10%) |
| 적격분할 | 요건 충족 여부 체크 + 과세이연 효과 산정 |
| 가업승계 | 600억 한도 적용 가능성 + 5년 사후관리 시뮬 |
| 지방세 | 과밀억제권역 중과 회피 입지 전략 |
| CFC | 해외자회사 실효세율 15% 판단 |
| 이전가격 | 로열티·관리용역료 arm's length 검증 |

### ④실행계획

**표준 타임라인:** → `report-template.md` §8 참조 (리포트 작성 시 이 타임라인을 시나리오에 맞춰 커스터마이즈)

참고 기간:
- 일반 지주전환: 6~12개월
- 금융지주: 12~18개월 (금융위 인가 추가)

### ⑤실행·모니터링

| 사후관리 | 기간 | 핵심 |
|----------|------|------|
| 적격분할 | 2년 | 사업계속·고용유지·지분유지 |
| 가업승계 | 5년 | 대표이사 종사·고용70%·지분유지 |
| NAV 할인율 | 상시 | 분기별 모니터링 + 배당정책 조정 |
| 내부거래 | 상시 | arm's length 정기 검증 |
| CP | 연1회 | 공정위 등급평가 대응 |

## 대형 컨설팅펌·회계법인 방법론 참고

| 유형 | 역할 | 강점 |
|------|------|------|
| MBB(McKinsey·BCG·Bain) | 전략 수립, 포트폴리오 최적화 | 산업 벤치마크, 전략 프레임 |
| Big4 회계법인(PwC·KPMG·EY·Deloitte) | 구조설계·세무·법무 실행 | 세무 시뮬레이션, 법률 실무 |
| 법무법인(김앤장·태평양 등) | 법률 자문, 규제 대응 | 공정위 대응, 소송 대비 |

한국 대기업은 통상 MBB(전략) + Big4(실행) 병행 활용.
