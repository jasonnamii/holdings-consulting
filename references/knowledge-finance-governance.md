## §1. 재무·자본구조

### 1-1. 지분구조 설계

| 유형 | 특성 | 장점 | 단점 |
|------|------|------|------|
| 피라미드형 | 소수자본→다층 지배 | 적은 자본으로 광범위 지배 | NAV 디스카운트 심화, 불투명 |
| 평면형 | 지주→자회사 직접 보유 | 투명, 이중과세 최소 | 지배 비용 큼 |
| 중간지주 활용 | 중간 레이어 삽입 | 규제 준수+경영 유연성 균형 | 구조 복잡성 증가 |

### 1-2. 지주사 4대 수익원

| 수익원 | 특성 | 세무 이슈 |
|--------|------|----------|
| 배당수입 | 주수익원, 자회사 수익성 의존 | 익금불산입 적용 (지분율별) |
| 브랜드 로열티 | 안정적 계약 기반 | 이전가격 심사 대상 |
| 경영자문료 | 서비스 실체 입증 필요 | 부가가치세 과세 |
| 임대수입 | 부동산 보유 시 | 취득세·재산세 |

### 1-3. 자금조달 수단

| 수단 | 특성 | 활용 시점 |
|------|------|----------|
| CB(전환사채) | 주식 전환 가능, 희석 리스크 | 주가 상승 기대 시 |
| BW(신주인수권부사채) | 권리행사 후 사채 잔존 | 부분적 자본확충 |
| EB(교환사채) | 자회사 주식으로 교환 | 자회사 지분 매각 우회 |
| IPO | 지주사 직상장 | 대규모 자금 필요 시 |
| 유상증자 | 기존/신규 투자자 | 긴급 자금 필요 시 |

부채비율: 공정거래법 법적 상한 100%(자산 초과 불가). 실무 관례상 200% 내 관리.

### 1-4. NAV 디스카운트

**정의:** 지주사 시가총액 < 자회사 지분가치 합(NAV) — 구조적 저평가.

**한국 현황:** 주요 6개 지주사 평균 할인율 약 51% (2024년 기준, NH투자증권).
- 한화 63%, LG 63%, 두산 63%, SK 45%, CJ 40%

**원인:** 불투명 지배구조, 낮은 배당, 소수주주 경시, 이중과세 우려, 내부거래 불신

**해소 방안:**
1. 배당성향 40%↑ 유지 (안정적·지속적)
2. 지배구조 투명성 강화 (사외이사 확대, 정보공시)
3. 자사주 매입·소각
4. 밸류업 프로그램 참여 (정부 정책)

**NAV 할인율 계산:**
```
할인율(%) = (NAV - 시가총액) / NAV × 100
```

| 할인율 구간 | 해석 |
|------------|------|
| 50%↑ | 심각한 저평가 (한국 평균) |
| 30~50% | 중간 수준 |
| 10~30% | 적정 |
| 0~10% | 프리미엄 수준 (희귀) |

### 1-5. 배당 파이프라인 설계

```
사업자회사 → 중간지주 → 최상위지주 → 주주
```

각 단계에서 익금불산입률 적용. 지분율 50%↑ 유지 시 파이프라인 전구간 비과세 가능. 2025년 배당소득 분리과세 시행으로 고배당 전략 유인 증가.

### 1-6. 내부거래 규제

- 공정거래법 §45①⑨: 부당지원행위 금지
- 상당히 낮거나 높은 대가로 자금·자산·용역 제공 금지
- 거래상 실질적 역할 없는 회사를 매개로 한 간접 지원 금지
- 판단 기준: 독립기업 간 거래가격과 비교, 경쟁력 집중 우려 여부

## §2. 거버넌스·지배구조

### 2-1. 오너-전문경영인 모델

최적 모델: "Creative Owner + Dutiful Manager + Strong Board"
- 지주사 대표: 전략·비전·감시 (CEO 아님)
- 자회사 대표: 경영 일선 실행
- 이사회: 상호 견제·보완

### 2-2. 이사회 구성 의무

| 구분 | 사외이사 비율 | 비고 |
|------|-------------|------|
| 일반 상장사 | 이사 총수의 1/4↑ | 대규모 제외 시 |
| 대규모 상장사 | 3명↑ + 과반수 | 자산 2조↑ |
| 금융지주 | 3명↑ + 과반수(50% 초과) | 이사 15명 이내 |

### 2-3. 상법 398조 자기거래 규제

| 항목 | 내용 |
|------|------|
| 적용 대상 | 이사 자기거래 + 주요주주·특수관계인 거래 |
| 승인 요건 | 이사회 2/3↑ 사전 승인 |
| 사후 추인 | 불가 — 반드시 사전 |
| 위반 효력 | 상대적 무효 + 손해배상 책임 |
| 실무 | 중요사실 공개 + 이해관계자 의결권 배제 |

### 2-4. 컴플라이언스 (CP)

**공정거래 자율준수프로그램 — 2024년 6월 법제화**

8가지 필수 도입요건:
1. 자율준수관리자 + 주관부서 지정
2. 임직원 역할 정의
3. 자율준수편람 작성
4. 운영 실태 관리
5. 임직원 교육
6. 위반 시 제재 규정
7. 모니터링·개선 체계
8. 리스크 평가·대응

지주사는 그룹 전체 CP 총괄. 자회사는 독립 CP + 지주사 보고 이중 구조.
우수 CP 운영 시 과징금 감경 혜택.

### 2-5. 소수주주 보호

**주식매수청구권:**
- 발동: 지주전환 결의에 반대한 주주
- 기간: 총회 결의일부터 20일 이내
- 가격 결정: 법정가격 → 협의 → 법원 판정("공정한 가액")

**95%↑ 지배 시:** 소수주주 강제매수(매도청구권) 가능

**컨설팅 시 필수 검토:**
- 소수주주 비율·활동성 사전 파악
- 주식매수청구 가격 시뮬레이션
- 소수주주 커뮤니케이션 전략 수립

### 2-6. ESG 거버넌스

| 영역 | 구체 내용 |
|------|----------|
| 이사회 다양성 | 성별·전공·경력 다양성 |
| 임원 보상 | ESG 지표 연동 (글로벌 스탠다드) |
| 윤리경영 | 반부패·공정거래 준수 |
| 공시 | K-ESG 가이드라인, 상장사 ESG 정보공개 |
