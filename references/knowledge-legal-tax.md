# 축1. 법률·규제 + 축2. 세무·조세

컨설팅 시 법률·세무 축은 항상 먼저 로드. "가능한가"(법률)와 "유리한가"(세무)를 동시 판단해야 구조 설계에 진입 가능.

---

## §1. 법률·규제

### 1-1. 지주회사 정의 (공정거래법)

**적용 기준 (AND 조건):**
- 자산총액 5,000억원 이상
- 자회사 주식가액 합계 ≥ 자산총액의 50%

> 5,000억 미만이면 공정거래법 지주회사 규제 적용 외. 규제 자유도가 크게 달라지므로 자산규모 진단이 최우선.

### 1-2. 의무 지분율

| 구분 | 상장 | 비상장 | 비고 |
|------|------|--------|------|
| 자·손자회사 | 30% | 50% | 2021.12 개정 기준 |
| 신규·편입 | 40% | 60% | +10%p 가산 |
| 증손회사 | 100% | 100% | 2025~26 첨단산업 50% 완화 추진 |

### 1-3. 설립·전환 방식

| 방식 | 핵심 메커니즘 | 주 활용 상황 | 세무 연계 |
|------|-------------|-------------|----------|
| 인적분할 | 기존 주주에게 존속+신설회사 주식 비례 배분 | 지주전환 시 선호 | 적격분할 요건 충족 시 과세이연 |
| 물적분할 | 기존회사가 신설회사 주식 100% 보유 | 사업부 독립화 | 소수주주 배제 논란 |
| 현물출자 | 자산을 자회사에 현물 투입 | 추가자금 없이 지분 확대 | 양도소득세 이슈 |
| 주식교환 | 자회사 주주에게 지주사 주식 교부 | 분할 후 지분 확대 | 교환비율 적정성 심사 |
| 주식이전 | 완전모회사 신설 + 기존 주식 이전 | 금융지주 전환 | 금융위 인가 필요 |

### 1-4. 공정거래법 행위제한 체크리스트

- [ ] 부채비율: 자산 초과 불가 (법적 상한 100% 이하, 실무 관례 200% 내)
- [ ] 비계열사 지분: 5% 이하
- [ ] 금융·비금융 겸영: 원칙 금지 (금융지주회사법 별도)
- [ ] 자회사 외 계열회사 주식 보유: 금지
- [ ] 계열사 간 채무보증: 금지
- [ ] 공정위 신고: 설립/전환 후 30일 내
- [ ] 자회사 지분율 미달 시: 2년 내 유예기간 후 처분/취득 의무

### 1-5. 금융지주회사 — 별도 규제 레이어

| 항목 | 내용 |
|------|------|
| 설립 요건 | 1+ 금융기관 지배 + 자산 5천억↑ + 금융위 인가 |
| 허용 자회사 | 은행·카드·증권·보험·캐피탈·자산운용 |
| 사외이사 | 과반수(50% 초과) 의무 |
| 최근 동향 | 2024~25 출자규제 일부 완화, 금융리스업 허용 검토 |

### 1-6. 실행 절차 타임라인

```
이사회 결의 → 분할계획서 → 주주총회(2/3↑ 특별결의) → 채권자보호(1개월) 
→ 분할기일 → 공정위 신고(30일) → 재상장
```
통상 6~12개월 소요. 금융지주는 금융위 인가 추가로 12~18개월.

## §2. 세무·조세

### 2-1. 수입배당금 익금불산입

지분율이 곧 세금. 경계값에서 급변하므로 "경계값 직상" 전략이 핵심.

| 지분율 | 익금불산입률 | 실질 세부담 | 전략 |
|--------|------------|-----------|------|
| 50%↑ | 100% | 배당금 비과세 | 최적 — 유지 |
| 30~50% | 80% | 20% 과세 | 50% 도달 위한 지분 확대 검토 |
| 10~20% (2025 신설) | 50% | 50% 과세 | 20% 도달 검토 |
| 10%↓ | 30% | 70% 과세 | 비효율 — 구조 재검토 |

*(2024년 법인세법 기준. 2025년 10~20% 구간 신설 예정)*

### 2-2. 연결납세제도

| 항목 | 내용 |
|------|------|
| 적용 요건 | 모회사가 자회사 지분 90%↑ 보유 (2024년 확대) |
| 장점 | 결손 자회사 있을 시 그룹 전체 세금 감소, 조세공정성 |
| 단점 | 복잡한 회계처리, 탈퇴 시 이연세액 추징 |
| 실무 포인트 | 지분 90~100% 자회사 포함 범위 확대 → 절세 기회 증가 |

### 2-3. 가업승계 특례 (상속세 및 증여세법 §30의6)

| 항목 | 내용 |
|------|------|
| 증여세 한도 | 600억원 (10억원 공제) |
| 세율 | 10% (과표 60억 초과분 20%) |
| 사후관리 | 5년 (대표이사 종사, 고용70% 유지, 지분 유지) |
| 위반 시 | 추징세액 + 연체료 3% |
| 지주회사 활용 | 지주사 주식의 가업승계 특례 적용 → 경영권 집중 + 절세 동시 |

### 2-4. 적격합병·분할 과세이연

**적격 요건 (모두 충족):**
1. 사업 계속성 — 피합병법인 사업 계속 영위
2. 고용유지 — 사후관리 2년 내 고용 유지
3. 지분 유지 — 지배주주 주식처분 제한

**인적분할 vs 물적분할 세무 차이:**

| 구분 | 인적분할 | 물적분할 |
|------|---------|---------|
| 양도소득세 | 적격 시 이연 | 적격 시 이연 |
| 증여세 | 지분감소→저가증여 기회 | 법적 증여 판단 가능성 |
| 전략 | 낮아진 주가에 자녀 증여 → 상속세 절감 | 자회사 100% 보유 → 향후 IPO 가능 |

### 2-5. 부당행위계산부인 (이전가격)

지주-자회사 간 거래 arm's length 괴리 시 세무당국 조정. 대상: 로열티, 관리용역료, 이자율, 자산양도. 이전가격 문서화 의무 강화 추세. APA(사전가격합의) 활용 권장.

### 2-6. CFC 규정 (해외자회사 보유 시)

| 적용 조건 | 내용 |
|----------|------|
| 지배 | 내국인 지분 50%↑ |
| 세율 | 실효세율 15%↓ |
| 실질 | 고정시설·실질사업 없음 OR 수동소득 |
| 과세 | 배당간주금액으로 산입 (실제 배당 불문) |

### 2-7. 기타 세무 체크리스트

- [ ] 지방세: 과밀억제권역 내 법인 설립 시 취득세·등록면허세 3배 중과
- [ ] 부가가치세: 경영관리용역은 원칙 과세 (면세 아님)
- [ ] 연결납세 탈퇴 시: 과거 이연세액 추징 주의
